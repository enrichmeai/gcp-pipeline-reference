## 🎯 QUICK REFERENCE

### Command Cheatsheet

**Setup Environment:**
```bash
# Option 1: Docker
cd blueprint/setup && docker-compose up -d

# Option 2: Airflow
./blueprint/setup/setup_airflow.sh

# Option 3: Python
pip install -r blueprint/setup/requirements-test.txt
```

**Run Tests:**
```bash
# All tests
./blueprint/run_full_tests.sh --full --coverage --report

# Unit tests only
pytest blueprint/components/tests/unit/ -v

# Integration tests
pytest blueprint/components/tests/integration/ -v
```

**Deploy to GCP:**
```bash
cd blueprint/tools/gcp
./preflight_check.sh
./setupanddeployongcp.sh
```

**Validate Deployment:**
```bash
python validate_deployment.py
bash validate_deployment.sh
```
